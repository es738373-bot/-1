Задание 1) Дана строка 'AaBbCcDd'. Используя срезы с шагом получите две строки: только с заглавными и только со
строчными буквами. Выведите их на экран.
2) Алгоритм на с++ : 1)Создать исходную строку.Используем стандартную библиотеку <string> и объявим строку вида "AaBbCcDd".(string)
              2)Выделить заглавные буквы.Проходя по символам строки, используем проверку функцией isupper() из библиотеки <cctype>, чтобы убедиться, что символ является заглавной буквой. Копируем заглавные буквы в отдельную строку.(isupper)
              3)Выделить строчные буквы.Аналогичным образом проходим по символам строки и проверяем, являются ли они строчными с помощью функции islower(). Строчные буквы собираем в отдельный контейнер.(islower)
              4)Вывести полученные строки на экран.Используя потоки ввода-вывода из библиотеки <iostream>, выводим результирующие строки.
3) Алгоритм на java: 1)Объявление исходной строки.Мы создадим строку с именем original, содержащую значение "AaBbCcDd".
                      2)Создание двух пустых строк для сбора результатов.Нам понадобятся две строки: одна для заглавных букв (capitals), вторая для строчных букв (lowercase).
                      3)Перебор символов исходной строки.Будем проходить по каждому символу исходной строки и проверять его регистр с помощью методов класса Character:(Character.isUpperCase(char ch) — проверит, является ли символ заглавным.  
Character.isLowerCase(char ch) — проверит, является ли символ строчным.)
                      4)Сбор символов в отдельные строки.Если символ соответствует нужному типу (заглавный или строчный), он добавляется в соответствующую строку.(+)
4)Временная сложность алгоритма на с++:Алгоритм работает за линейное время относительно длины входящей строки, так как нам приходится перебирать каждый символ ровно один раз. Таким образом, временная сложность составляет O(n), где n — длина исходной строки.
